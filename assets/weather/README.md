# 天气特效贴图

将以下 PNG 放入本目录后，地图天气会使用**动态贴图**渲染雨、雪、风等效果；未放入时使用程序绘制的粒子。

**当前已放置**：`rain_light.png`、`rain_dark.png`（来自 [OpenGameArt - Rain](https://opengameart.org/content/rain)，CC-BY 3.0，作者 raidensan）、`snowflake.png`（来自 [OpenGameArt - Snow Animation](https://opengameart.org/content/snow-animation)，CC-BY 3.0/4.0，作者 AntumDeluge）。`wind_leaf.png` 未包含，风效果仍使用程序绘制；可从下方推荐素材补充。

## 推荐素材（免费可商用）

### 1. Weather particle sprites（推荐）

- **来源**：https://emiemigames.itch.io/weather-particle-sprites  
- **授权**：可自由使用，无需署名（作者建议署名 Emi Emi Games）
- **下载**：页面点击 “Download” → Name your own price 可填 0 免费下载

下载解压后，将下列文件**重命名**并放入本目录：

| 压缩包内文件名     | 重命名为（本目录） |
|-------------------|--------------------|
| `rain light.png`  | `rain_light.png`   |
| `rain dark.png`   | `rain_dark.png`    |
| `snowflake.png`   | `snowflake.png`    |
| `autumn leaf.png` | `wind_leaf.png`    |

### 2. Weather Effects Assets Pack（像素风）

- **来源**：https://free-game-assets.itch.io/weather-effects-assets-pack-pixel-art  
- 含雨、暴雨、风、雪、闪电等精灵图，可按需选取并重命名为上述四个文件名之一使用。

### 3. OpenGameArt

- **雨**：https://opengameart.org/content/rain （CC-BY 3.0）  
- **雪**：https://opengameart.org/content/snow-animation （CC-BY 3.0/4.0）  

使用前请遵守对应页面的授权说明。

## 本目录所需文件列表

| 文件名           | 用途     | 可选 |
|------------------|----------|------|
| `rain_light.png`  | 雨粒贴图 | 是   |
| `rain_dark.png`  | 雨粒贴图（深色） | 是   |
| `snowflake.png`  | 雪花贴图 | 是   |
| `wind_leaf.png`  | 风/落叶贴图 | 是   |

任意一个或多个放入即可生效；未放入的类别会使用程序绘制的粒子。
